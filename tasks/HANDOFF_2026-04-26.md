**요청**: "Insight에 들어오면 카테고리가 보이고, 그 안으로 들어가서 내용을 볼 수 있게 만들면 좋겠어"

**설계**:
- `/insights` → 4개 카테고리 카드 (현재는 8개 아티클 그리드 → 리디자인 필요)
- `/insights/semiconductor-parts` → 해당 카테고리 아티클 목록
- `/insights/esc` → 아티클 상세 (현재 동작 유지)

**구현 방법**:
- `src/app/insights/page.tsx` → 4개 카테고리 카드로 리디자인
- `src/app/insights/[slug]/page.tsx` → slug가 카테고리 슬러그면 아티클 목록, 아니면 아티클 상세 렌더링

**카테고리 슬러그 목록**:
```
CATEGORY_SLUGS = ["semiconductor-parts", "laser-equipment", "thermal-management", "ev-charging"]
```

**카테고리 → 아티클 매핑** (article.category 필드 기준):
- `semiconductor-parts`: esc, wafer-carrier, dry-vacuum-pump, oring
- `laser-equipment`: laser-equipment, waterjet-laser
- `thermal-management`: thermal-management
- `ev-charging`: ev-charging

**generateStaticParams** — 카테고리 슬러그 + 아티클 슬러그 모두 포함해야 함

**주의**:
- 서버에 `/insights/[category]` 폴더가 예전에 존재했다가 삭제됨 (이번 세션에서 `rm -rf`로 제거)
